Memory game

This project is a game to demonstrate JavaScript language skills.

Game rules:

You need to click on cards with icons to open them. The game doesn't allow the player to open more than two cards simultaneously. If the cards have the same image, they will remain open; otherwise, they will flip back. The goal of the game is to open all the cards in the shortest time possible.

The speed rating is evaluated with stars in the top left corner.

To play game: https://Rdev2023.github.io/memory-game/


The game is based on: Javascript Arrays, Map, setTimeout() function.

GitHub source code: https://github.com/Rdev2023/memory-game

Venues

This single-page application provides the user with a list of highly-rated venues in the selected area.

The venues are presented to the user with detailed information from the FourSquare API, including menus, locations, opening hours, and ratings.

To select an area, a search bar with a dynamic dropdown list is used, which loads information from the Google API. The user is free to search for a location based on geographical location, street name, districts, cities, or zip codes. Once the user selects a location, it is displayed on a map from the Google API with markers for all the establishments in that area.

If this description seems complex, a short video attached easily and clearly demonstrates the functionality of the application.

Details: https://github.com/Rdev2023/Venues/README.md

GitHub source code: https://github.com/Rdev2023/Venues

The project based on: Google API, FourSquare API, React, Redux
